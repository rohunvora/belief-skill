# Belief Router

You already have beliefs about the world. You just don't realize they're trades.

*"Everyone's on Ozempic." "AI is going to replace search." "This Trump stuff is jumping the shark."*

These aren't just observations ‚Äî each one is a directional bet. The problem isn't conviction. It's that you've never connected your worldview to the instruments that let you profit from it.

**Belief Router is an [OpenClaw](https://github.com/openclaw/openclaw) skill that turns what you think into what you trade.** Voice note, text, pasted tweet ‚Äî say what you believe and get the single best way to express it.

## How It Works

1. **Finds the deeper claim** ‚Äî *"everyone's on Ozempic"* ‚Üí the play isn't pharma (consensus), it's shorting food delivery (the victim)
2. **Checks prediction markets first** ‚Äî if Kalshi or Polymarket has a direct contract, everything else has to beat 100% thesis beta
3. **Scores on one metric** ‚Äî `thesis beta √ó convexity / (1 + time cost)` ‚Äî no gut feel, no defaults
4. **Structures the position** ‚Äî direction theses decompose into independent bets (partial wins > all-or-nothing)
5. **One trade, one screen** ‚Äî scenario table, kill conditions, conviction breakeven, alt from a different class

Searches Kalshi, Polymarket, Robinhood, Hyperliquid, Bankr, and private markets automatically. You don't pick the platform.

## Paper Trading

Not ready for real money? Every trade card has a üìù **Paper Trade** button. Track beliefs like a portfolio.

```bash
bun run scripts/track.ts portfolio    # all open beliefs + live P&L
bun run scripts/card.ts --id <ID>     # shareable trade card
```

## Install

Requires [Bun](https://bun.sh) and [OpenClaw](https://github.com/openclaw/openclaw).

```bash
git clone https://github.com/rohunvora/belief-skill.git
cd belief-skill/v2/belief-router
bun install
```

Point your OpenClaw skill path to `v2/belief-router`. Activates automatically when you express a belief ‚Äî no commands, just talk.

No API keys required. Uses public APIs (Yahoo Finance, Polymarket, Kalshi, DexScreener).
